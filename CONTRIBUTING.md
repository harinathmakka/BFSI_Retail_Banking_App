# Contributing

Thanks for your interest in the BFSI Retail Banking App!

## Quick rules
- Fork the repo and create feature branches.
- Use clear, focused commits. Use prefixes like `feat:`, `fix:`, `docs:`, `chore:`.
- Do not commit secrets or `.env` files.

## How to run locally
1. Copy `backend/.env.example` -> `backend/.env` and fill values.
2. Run `bash scripts/setup.sh`
3. Start backend and frontend as described in `bash scripts/run-dev.sh`.

## Pull request checklist
- Code builds and runs locally.
- No sensitive data in the PR.
- Include a short description of changes and any required steps to test.
